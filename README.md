Exploratory Data Analysis (EDA) on Retail Sales Data

Project Overview

This project performs Exploratory Data Analysis (EDA) on Retail Sales Data to identify trends, patterns, and business insights. The analysis helps understand sales performance, customer behavior, and product demand.

Objectives

* Clean and preprocess retail sales data
* Analyze monthly sales trends
* Identify top-performing product categories
* Study regional sales performance
* Visualize insights using graphs

Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

Dataset Features

* Order Date
* Customer ID
* Product Category
* Quantity
* Unit Price
* Total Amount
* Region

Analysis Performed

✔ Data Cleaning
✔ Handling Missing Values
✔ Date Conversion
✔ Monthly Sales Analysis
✔ Category-wise Revenue
✔ Data Visualization

Key Insights

* Identified peak sales months
* Found highest revenue generating categories
* Observed seasonal trends in retail sales

How to Run

1. Download the dataset
2. Install required libraries:
   pip install pandas numpy matplotlib seaborn
3. Open the Jupyter Notebook and run all cells
